Multiplication Grid

Break big numbers into small pieces — every cell is a times-table fact

How it works

  1. Enter X (up to 4 digits) and Y (up to 2 digits).
  2. X is broken by place value: K=1000, H=100, T=10, U=1.
  3. Each grid cell = one digit of (X-piece × Y-piece).
  4. Add each column downward to get the SUM row.
  5. Copy each SUM into the assembler at the correct place value.
  6. Add all Row Totals to get the Final Answer.
